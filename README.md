# Introduction
The current project is creating a dead simple recipes website with basic HTML elements.

# Skills
The following skills are needed in order to complete the project:
- Basic HTML elements such as paragraphs, headers, lists, links, and images.
- Basic git workflow such as cloning, staging, committing, and pushing.

# About
This project is part of [The Odin Project](https://www.theodinproject.com/)'s foundation course. You can find the link to the corresponding lesson [here](https://www.theodinproject.com/lessons/foundations-recipes).
